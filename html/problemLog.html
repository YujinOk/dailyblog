How to push git cloned repository (from someone) to your own on GitHub

The repo still has remote repo from the original repo so you need to first remove that

  1. go to terminal
  2. run [git remote -v] to check and see the remote repos
  3. run [git remote remove origin]
  4. then run [git remote add origin https:newlink]
  5. run [git branch -M main]
  6. git status- git add . - git commit -m ""
