Adding react projects to Github Pages

  1. Run this command in the main directory
    npm install gh-pages --save-dev
  2. Open the package.json file and add the following 'homepage' property:
    "homepage": "http://{githubusername}.github.io/{reponame}"
  3. In the scripts property add:
    "scripts": { //... "predeploy": "npm run build", "deploy": "gh-pages -d build" }
